Webbpubliceringssystem




PHP - Hypertext Preprocessor

PHP är ett skriftspråk som används på webbservrar för att driva internethemsidor med
dynamiskt innehåll. Alltså innehåll som genereras från till exempel besökarens data eller
en databas. Språket gavs ut 1995 av Rasmus Lerdorf men har uppdatterats ett antal gånger.
Php är fortfarande ett populärt skriftspråk idag.





Javascript

Javascript är ett annat webbpubliceringssystem och används också idag.
Det används främst för att skapa skripter som körs på en hemsida.
Men det kan implementeras i program som inte är i en webbläsare.
Ett exempel på javascript utanför hemsidor är Node.js, som tillåter
utvecklaren att använda JavaScript på serversidan för att kunna arbeta med
anslutningar till databaser, skicka e-post-meddelanden och så vidare.
